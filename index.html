Filosofía

Tus notas no deberían pertenecer a una app

La mayoría de apps de productividad guardan tus datos en sus servidores, en formatos propietarios. Si la app cierra, tus notas desaparecen. From funciona al revés.

Archivos .md estándar

Cada nota es un archivo Markdown en tu disco. Puedes abrirlo con VS Code, Obsidian, iA Writer o cualquier editor de texto. Ningún formato propietario.

En tu carpeta, no en la nube

Tus datos viven en una carpeta de tu Mac (por defecto en iCloud Drive). From lee esa carpeta. Si desinstalas From, tus notas siguen ahí.

From es la capa, no el contenedor

From añade productividad — vistas, tareas, IA, automatización — sobre tus archivos. Es una capa inteligente, no una prisión para tus datos.

Preguntas frecuentes

¿Necesito Obsidian para usar From?

No. From es completamente independiente. Usa archivos Markdown estándar, así que puedes abrir las mismas notas con Obsidian, iA Writer o cualquier editor si quieres, pero no es necesario.

¿Cómo funciona la IA?

From indexa tus notas localmente. Cuando preguntas algo, recupera los fragmentos relevantes y los envía a la API de IA. Solo el fragmento necesario viaja — nada se almacena en servidores externos. Puedes elegir entre Claude, GPT o Gemini.

¿Puedo sincronizar entre dispositivos?

Sí. Si usas iCloud Drive (opción por defecto), tus notas se sincronizan automáticamente entre todos tus Macs. También puedes vincular notas a Google Docs para colaborar.

¿Funciona en iPhone o iPad?

De momento From es solo para macOS. Tus notas en iCloud son accesibles desde cualquier editor de texto en iOS. La app nativa para iOS está en desarrollo.

¿Qué pasa si dejo de usar From?

Nada. Tus notas son archivos .md en tu ordenador. Siguen ahí, legibles con cualquier editor de texto. No hay formato propietario ni dependencia alguna.

¿Necesito una API key para la IA?

No necesariamente. Puedes usar la IA gestionada de From (con suscripción) o aportar tu propia API key de Anthropic, OpenAI o Google. También puedes conectar tu suscripción de Claude directamente.

¿Mis datos están seguros?

Tus notas nunca salen de tu Mac ni de tu iCloud privado. From no tiene servidores que almacenen tus datos. No hay telemetría, no hay analíticas, no hay tracking.
